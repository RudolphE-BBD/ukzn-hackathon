Hackathon

Theme

Childhood games (with a twist)

Brief

Recreate a beloved childhood game and give it a twist.

Objective

Build and deliver a fully featured PWA using HTML, CSS and Javascript.

Limitations/Restictions/Rquirements

  • Final submissions at 18:00, Saturday, 12 November 2022 (link to submission form will be posted)
  • Submission webinar at 17:30 on Saturday for those who need help submitting
  • The web application has to be live has to be publicly accessible
  • The source code has to be delivered in a public git repo
  • No late submissions!

Scoring (0-5)

  • Playabilty
  • Code quality
  • Execution (how far are you 50% done, 100% done?)
  • Conformance to the objective
  • Creativity
